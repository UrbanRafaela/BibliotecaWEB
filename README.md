# Sistema de Biblioteca

Projeto desenvolvido com Django e PostgreSQL.

## Funcionalidades

* Login de usuários
* Controle de permissões
* Cadastro de livros
* Edição de livros
* Exclusão de livros
* Pesquisa de livros
* Upload de capa dos livros
* Interface responsiva com Bootstrap

## Tecnologias Utilizadas

* Python
* Django
* PostgreSQL
* Bootstrap
* Pillow
* Psycopg2
